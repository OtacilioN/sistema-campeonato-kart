# Design System Usage

Claude Design artifacts live in `references/design-system/`.

Use them as visual reference for layout, spacing, color, typography, and flow. Do not treat the mockup JSX files as production code.

The base visual language is a light sporting dashboard: red, black, white, gray surfaces, compact cards, tabular numbers, bottom navigation and small racing accents.

Preserve the approved visual direction. Future changes should be incremental and justified by product needs.
