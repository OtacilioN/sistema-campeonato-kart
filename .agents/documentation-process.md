# Documentation Process

This project publicly documents its creation with AI.

Preserve `docs/documentacao-processo.md` as historical record. Do not rewrite the transcript or remove process context.

Use:

- `CONTEXT.md` for domain glossary only.
- `docs/adr/` for hard-to-reverse decisions with real trade-offs.
- README for setup and operational instructions.

When an agent makes an important decision, record it in the task summary or in an ADR if it meets the ADR threshold.
