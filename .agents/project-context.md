# Project Context

**Velocidade Quase Máxima** is a mobile-first PWA for an amateur kart championship. **Velozes** is the internal codename.

Production domain: `velocidadequasemaxima.com.br`.

The app shows public ranking, calendar, pilots, pilot profiles, battery details and lap-to-lap analysis. Admin users confirm battery results through a password-protected area.

The championship uses seasons, batteries, pilots, battery results, discard rules, tie-breakers and public lap-to-lap validation. Read `CONTEXT.md` before modeling these concepts.

The project is open source under MIT, hosted on GitHub, deployed to Vercel Hobby and backed by Neon PostgreSQL through Prisma.
