# Resultado confirmado como fonte de verdade

O PDF oficial e a insercao manual sao fontes de entrada para uma revisao administrativa, mas o ranking, os perfis e o historico publico usam apenas o resultado confirmado. Essa decisao permite corrigir falhas de parse, marcar pole position, aplicar penalizacoes e revisar dados antes de alterar a classificacao publica, sem armazenar o PDF bruto como verdade final.
