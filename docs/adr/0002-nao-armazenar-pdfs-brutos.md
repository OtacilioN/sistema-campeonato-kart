# Nao armazenar PDFs brutos

Os PDFs de resultado oficial e lap-to-lap sao processados apenas como entrada temporaria, e o sistema persiste somente os dados estruturados extraidos e confirmados ou validados. Essa decisao reduz custo, privacidade, complexidade de storage e dependencia de infraestrutura no plano Vercel Hobby, aceitando que auditoria por arquivo original nao faca parte do MVP.
